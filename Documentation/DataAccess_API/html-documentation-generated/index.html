post /CovidDataBase
Solicita una búsqueda determinada en base de datos. (post)

Obtiene información relativa a la evolución de la pandemia dependiendo del elemento method requerido. Las posibilidades son:

  • GetGeoZoneData. Obtiene los datos para una o varias zonas geográficas dentro de un rango de fechas.
  • GetAllGeoZoneData. Obtiene los datos para todas las zonas geográficas disponibles dentro de un rango de fechas.
  • GetAllGeoZoneForAllDates. Obtiene todos los datos disponibles en base de datos.
  • GetAllCountries. Obtiene una lista de paises disponibles para consulta.
  • GetAllDates. Obtiene una lista con todas las fechas disponibles para consulta.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

InputPOST InputPOST (optional)
Body Parameter — Método invocado y descriptivo con zona o zonas geográficas solicitadas más rango de fecha.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Consulta exitosa. Ver Models/GeoZone.

400

Solicitud no válida.
